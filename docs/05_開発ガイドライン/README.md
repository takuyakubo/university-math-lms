## 開発チーム体制

- プロジェクトマネージャー (PM): 全体の進行管理と調整
- フロントエンド開発者: Next.js、TypeScriptを中心とした実装
- バックエンド開発者: Python/FastAPI、データベース設計を中心とした実装
- UIデザイナー: ユーザーインターフェースのデザイン
- QAエンジニア: テスト計画と実行
